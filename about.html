Our Mission

ShopEase was created as part of our IMT 405 Web Application Development capstone project. Our goal is to demonstrate a simple, functional e-commerce product listing platform that allows users to browse, search, filter, and add products to a shopping cart — all using core HTML, CSS, and JavaScript with no external frameworks.

# ShopEase

## Project Description
ShopEase is a simple e-commerce product listing web application built for the
IMT 405 Web Application Development Capstone Project. It allows users to browse
a catalog of products, search and filter by category and price, sort results,
add items to a shopping cart, adjust quantities, and "checkout" (demo only —
no real payment processing). The site also includes a Team
page and a Contact page with form validation and an FAQ accordion.

**Option Chosen:** Option A — Core Web Technologies (HTML, CSS, JavaScript only — no frameworks)

## Technologies Used
- HTML5 — semantic page structure
- CSS3 — custom styling, Flexbox, Grid, responsive media queries
- Vanilla JavaScript (ES6) — interactivity, cart logic, filtering, form validation
- LocalStorage — persists cart contents between page loads
- Local image assets — team photographs and site imagery

## Setup / Installation Instructions
No build tools or server required — this is a static site.

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox).

To run locally with a simple server (optional, recommended for consistent
relative-path behavior):
```bash
# Using Python
python3 -m http.server 8000
# Then visit http://localhost:8000
```
